## About The Project

&nbsp;
<div align="center">
    <img src="https://github.com/Anthonyr06/Messaging-System/blob/master/server-on.png" alt="Logo" width="350">&nbsp;&nbsp;&nbsp;
    <img src="https://github.com/Anthonyr06/Messaging-System/blob/master/client-on.png" alt="Logo" width="450">
</div>
&nbsp;

&nbsp;
MessagingSystem is a WPF-MVVM instant messaging application that uses Sockets for real-time communication. It enables the user to send text messages to other clients that are connected to and logged into the chat server. 
The project consist in two WPF apps: The Client and The Server, where both interacts together to form the Messaging System.


&nbsp;
### Built With

* WPF
* C#
* XAML

&nbsp;
&nbsp;

## Getting Started

### Prerequisites

* .Net 6.0
* Visual Studio 2019/2022

&nbsp;
&nbsp;
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Anthonyr06/Messaging-System.git
   ```
2. Run it on Visual Studio
&nbsp;
&nbsp;

## Usage


1. Run ChatServer project
2. Click 'Start the server' Button

    <img src="https://github.com/Anthonyr06/Messaging-System/blob/master/server-off.png" alt="Logo" width="200">
    
3. Run ChatClient project
4. At the left upper corner, write down an username
5. click login button
  
    <img src="https://github.com/Anthonyr06/Messaging-System/blob/master/client-off.png" alt="Logo" width="200">
  
6. Open another instance of ChatClient project
7. Redo steps 4 and 5
8. Done!
